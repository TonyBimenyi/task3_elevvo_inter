# 🌦️ Real-Time Weather Dashboard

A minimal and responsive weather dashboard that fetches real-time weather data using the **OpenWeatherMap API**.  
Easily search for multiple cities, view current conditions, and check a 3-day forecast.

---

## ✨ Features
- 🔍 **Search Cities** – fetch weather data by city name.  
- 🌡️ **Current Weather** – temperature, description, and weather icons.  
- 📅 **3-Day Forecast** – shows midday forecasts.  
- ⏳ **Loading States** – visual feedback while fetching data.  
- 📍 **User Location Support** – auto-detect and display your local weather.  
- 🎨 **Clean & Minimal UI** – dark mode inspired design.  

---

## 🛠️ Tools & Libraries
- **HTML5, CSS3, JavaScript**
- **OpenWeatherMap API**
- **Fetch API / Axios**
- **DOM Manipulation & Asynchronous Operations**
